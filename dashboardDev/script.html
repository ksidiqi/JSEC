Instruction

Inorder to upload errors from clinet-side, you must include this java script right after <head> tag and the html form after the <body> tag. Moreover, replace your email and the group name in the the html inorder to have the errors to be registered in the database. If you are not part of a group please replace 'your group' with 'none'






<script> 
    window.onerror = function (errorMsg, url, lineNumber) 
    {
        var er =document.getElementById('error');
        var ur = document.getElementById('url');
        var li = document.getElementById('line');
        var form = document.getElementById('hidenform');
        er.value = errorMsg;
        ur.value = url;
        li.value = lineNumber;
        form.submit();

    }
</script>
                        

                                
<form id="hidenform" method="POST" target="hidden-form-iframe" action="http://104.131.199.130/submiterror.php">  
    <input type='hidden' name='error' id='error' />
    <input type='hidden' name='url' id='url'/>
    <input type='hidden' name='line' id='line'/>
    <input type='hidden' name='group' id='group' value='your group'/>
    <input type='hidden' name='email' id='email' value='your email'/>
</form>
